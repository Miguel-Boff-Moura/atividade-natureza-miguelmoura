# Natureza

Este projeto consiste em uma página web desenvolvida em HTML e CSS com o tema **Natureza**. O objetivo é apresentar informações sobre árvores brasileiras, notícias relacionadas ao meio ambiente e um formulário de contato.

## Funcionalidades

- Cabeçalho com menu de navegação;
- Cards com informações sobre árvores brasileiras;
- Área lateral com notícias ambientais;
- Formulário de contato com validação de campos HTML;
- Rodapé com informações da empresa e redes sociais.

## Tecnologias utilizadas

- HTML5
- CSS3

## Como executar

1. Baixe ou clone o projeto.
2. Certifique-se de que a pasta `img` e o arquivo `style.css` estejam no mesmo diretório do `index.html`.
3. Abra o arquivo `index.html` em qualquer navegador.
